Online Bus Ticket Reservation System is a Web based application that works within a centralized network. This project presents a review on the web application "Orient bus Transport system" as should be used in a bus transportation system, a facility which is used to reserve seats, cancellation of reservation and different types of route enquiries used on securing quick reservations. Orient bus transport system is a web application works within centralized network. This project provides an effective solution for managing bus ticket information using a database. Our system has three login for user, admin and conductor. This system provides web application as well as android application for people to get their Bus ticket online. This system provides functionality like accessing basic information of user for authentication and provide bus ticket for the user without placing them in long queues. The conductor in bus would be able to verify the ticket by scanning the QR code provided on the ticket with are commended device. The notification generated by the system would be send to the user in form of message and PRN no. such as when where and what time the card was use. This system also provides online payment facility.
